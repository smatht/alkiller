La idea:
--------
Se trata de desarrollar un software pensado para administrar alquileres de inmuebles (como departamentos, casas, cabañas, etc.) con todas las funcionalidades básicas para un eficiente desarrollo de la actividad y comunicación con los inquilinos, con guías visuales para nuevos usuarios.

Objetivos:
----------
* Desarrollar un Trabajo Final de Aplicación para la creación de un software de administración de inmuebles y mejorar con el mismo la comunicación propietario / inquilino
* Implementar eficientemente el sistema de administración de inmuebles en una nube pública
